Highlight Tahun Anomali

Tahun 2021–2022

Lonjakan harga dipengaruhi pandemi COVID-19 dan cuaca ekstrem yang mengganggu produksi serta distribusi pangan nasional.

Analisis Kenaikan dan Penurunan Harga

  • 2019–2020: Harga relatif stabil.
  • 2020–2021: Kenaikan akibat pandemi.
  • 2022: Lonjakan karena cuaca ekstrem.
  • 2023–2025: Harga mulai stabil.

Perbandingan Antar Sayuran

  • Cabai Merah: Volatilitas tinggi.
  • Bawang Merah: Dipengaruhi biaya produksi.
  • Tomat: Fluktuatif namun cepat panen.
  • Kentang: Relatif stabil.

Insight Utama

Kesimpulan Data

Cabai merah menunjukkan volatilitas harga tertinggi, sedangkan kentang relatif stabil karena daya simpan lebih lama.

Tools dan Teknologi

  • HTML & CSS
  • JavaScript
  • Chart.js (visualisasi data)
  • Data publik (BPS, Kementerian Pertanian, PIHPS)

Refleksi Proyek

Melalui proyek ini, saya belajar mengolah data harga pangan menggunakan sumber data publik dan menyajikannya dalam bentuk visualisasi grafik untuk mengidentifikasi pola perubahan harga sayuran dari waktu ke waktu.

Proyek ini saya kerjakan secara mandiri, sehingga membantu saya memahami keterkaitan antara data, pemrograman, dan permasalahan nyata di bidang pangan dan pertanian.

Pengalaman ini menumbuhkan ketertarikan saya terhadap bidang komputasi dan pengolahan data, khususnya dalam penerapannya untuk analisis sektor pertanian di jenjang pendidikan selanjutnya.

Kesimpulan

Proyek ini menunjukkan bagaimana data sederhana dapat dimanfaatkan untuk memahami dinamika sektor pangan di Indonesia.

Visualisasi data membantu mengungkap pola tren dan anomali harga yang tidak mudah terlihat melalui pengamatan kasat mata.
